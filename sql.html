Sql

La historia de SQL comienza en 1969, cuando el investigador de IBM Edgar F. Codd definió el modelo de base de datos relacional. Ese modelo se basa en la asociación de «claves» con varios datos. Por ejemplo, un nombre de usuario se puede asociar con un nombre real y un número de teléfono. Unos años más tarde, IBM creó un lenguaje para los sistemas de gestión de bases de datos relacionales basado en el trabajo de Codd. Ese lenguaje primero se llamó SEQUEL, siglas de «Structured English Query Language» y después de varias implementaciones y revisiones, pasó a llamarse SQL.
